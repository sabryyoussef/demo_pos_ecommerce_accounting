3. PIN cashiers — no backend user required

Key message: Shift cashiers are employees only (hr.employee). They select their name on the POS and enter a PIN. They never receive an Odoo user, email, or backend password — reducing licences, IT overhead, and security risk. Managers use admin (or POS Manager) in the backend to see who sold what, where.

  1. Click Open POS for a shop (table below).
  2. Choose the employee name on the till login screen.
  3. Enter the PIN.
  4. On Opening Control, tap Open Register (0.00 is fine).
  5. Sell → pay. Admin reviews Point of Sale → Orders by employee and shop.
